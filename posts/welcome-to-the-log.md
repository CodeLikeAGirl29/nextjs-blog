This is entry zero. From tomorrow on, a GitHub Actions workflow runs every morning, pulls a quote from a public API, writes a new markdown file into `/posts`, and commits it straight to this repository.

No human touches the files after this one. Open `/scripts/generate-post.mjs` to see exactly how each entry gets made.
